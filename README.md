# Data Visualization (Python, Matplotlib, Seaborn)

## Project Overview
This project focuses on exploratory data analysis and visualization of a real estate dataset using Python.  
The goal is to identify patterns in property characteristics such as price, area, and construction year through visual analytics.

## Objectives
- Perform data visualization using Matplotlib and Seaborn  
- Analyze relationships between property features  
- Identify trends across price segments  
- Customize visual styles for analysis and presentation  

## Dataset Description
The dataset contains real estate data, including:
- construction_year  
- house_area  
- price  
- subtype_of_property  
- type_of_property  
- locality  


## Key Visualizations

### Scatter Plot (Matplotlib)
- Relationship between construction year and house area  
- Helps identify trends in property size over time  

### Line Chart (Matplotlib)
- Average house area by price groups (quantiles)  
- Shows how area changes across price segments  

### Bar Chart
- Distribution of properties by subtype  
- Highlights the most common property types  

### Lineplot (Seaborn)
- Trend of average area across price groups  

### Scatterplot (Seaborn)
- Area vs construction year with property type segmentation  

### Countplot
- Distribution of property types over time  

### Boxplot
- Price distribution across property types  

### KDE Plot
- Distribution of house area for top locations  

## Styling & Context
- Used Seaborn `set_style` and `set_context`  
- Adjusted for both notebook analysis and presentation-ready visuals  

## Skills Demonstrated
- Python (Pandas, Matplotlib, Seaborn)  
- Data visualization  
- Exploratory Data Analysis (EDA)  
- Data grouping and transformation
